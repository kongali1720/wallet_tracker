📊 Track banyak wallet dan tampilkan saldo & histori via terminal
# wallet_tracker

Script untuk memantau saldo ETH dari banyak wallet sekaligus.

---

## Cara pakai

1. Install dependensi: `pip install -r requirements.txt`
2. Masukkan daftar wallet di `wallets` list di `main.py`
3. Jalankan: `python main.py`
